# 나만의 퀴즈 게임 🎯

터미널에서 동작하는 파이썬 기반 퀴즈 게임입니다. 입력이나 예외 처리 과정이 적용되어 부드러운 게임 경험을 제공합니다. 

## 프로젝트 개요
본 프로젝트는 Python 기초, 객체 지향 프로그래밍(클래스형), 파일 입출력(`json` 모듈)을 학습하고, Git 명령어 활용법을 체득하기 위해 진행되었습니다. 객체 지향 설계를 위해 퀴즈 데이터를 담는 `Quiz` 객체, 메인 루프와 상태 관리를 위한 `QuizGame` 객체 형태로 분리했습니다. 

## 퀴즈 주제 선정 이유
영화(타노스, 인터스텔라, 기생충, 해리포터, 매트릭스) 주제를 차용했습니다. 이유는 다양한 사람들이 즐겁게 문제를 풀 수 있는 보편적인 대중문화를 반영하기 때문입니다.

## 실행 방법
1. 로컬 환경에 파이썬이 설치되어 있는지 확인합니다. (Python 3.10 이상 권장)
2. 터미널을 열고, 프로젝트 폴더 경로로 이동합니다.
3. 다음 명령어를 실행합니다.
   ```bash
   python main.py
   ```

## 기능 목록
1. **퀴즈 풀기**: 문제 수를 직접 선택하고 저장된 퀴즈를 섞어(Random) 정답을 맞힙니다. 문제당 기본 배점은 10점이며, 풀이 중 '0'번을 입력하여 힌트를 사용할 경우 5점이 감점(5점 획득)됩니다.
2. **퀴즈 추가**: 사용자가 직접 문제, 보기(4개), 정답 번호 및 힌트를 입력하여 퀴즈를 생성할 수 있습니다. 
3. **퀴즈 목록**: 등록된 모든 퀴즈의 문제 제목 리스트를 출력합니다.
4. **퀴즈 삭제**: 삭제할 퀴즈의 번호를 선택하여 불필요한 문제를 영구적으로 제거합니다.
5. **점수 및 이력 확인**: 최고 점수와 더불어 플레이어가 진행한 최근 5번의 퀴즈 히스토리(날짜/시간, 푼 문제수, 점수)를 확인합니다.

> ⚠️ 잘못된 입력(문자 등), 빈 칸, `Ctrl + C` 누락 등의 이벤트가 발생할 경우 게임이 비정상 종료 되지 않고, 데이터 저장 후 안전하게 종료 혹은 안내 메시지를 출력합니다.

## 파일 구조
- `main.py`: 프로그램 주요 진입점 (메뉴 호출).
- `quiz.py`: 단일 퀴즈 클래스 데이터 모델 (`Quiz`).
- `game.py`: 퀴즈 게임 로직 총괄 및 `state.json`을 통한 데이터 접근 객체 제공 (`QuizGame`).
- `data.py`: `state.json`이 존재하지 않을 시 제공하기 위한 최초 기본 퀴즈 5종 포함 로직.
- `state.json`: 최고 점수 및 등록된 퀴즈 딕셔너리 정보가 보관되는 파일 (최초 실행 후 자동 생성됨).

## 데이터 파일 설명
- **경로**: 프로젝트 최상단 디렉토리 (`state.json`)
- **역할**: 게임 진행 시 추가한 새로운 퀴즈 내용과 플레이어의 최고 점수를 반영구적으로 보존하기 위해 사용됩니다.
- **스키마 구조**(예시):

```json
            "date": "2026-04-09 18:05:01",

```
